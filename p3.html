PosePro: Real-time Posture Analysis Tool

Key skills & Tools: HTMl | CSS | JavaScript | Machine Learning | Python

PosePro is an online platform that leverages the MoveNet model to recognize and analyze body postures through real-time data collected from the user's camera. It provides immediate feedback on posture accuracy and duration held correctly, helping users improve their posture and overall body alignment. The platform also tracks progress over time, allowing users to see their improvement journey.
